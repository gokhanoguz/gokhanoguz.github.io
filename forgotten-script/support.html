How the game works

What is a cipher run?

Each run uses one randomly-generated substitution alphabet. You see a series of inscriptions encoded with that alphabet, starting easy and ending with a famous-quote finale. Decoding any letter in one inscription decodes it everywhere, so the run gets easier as you progress.

Daily vs endless runs

The daily run uses a seed that's the same for every player on a given UTC day, so streaks and leaderboards are comparable. Endless runs use a fresh random seed and don't affect your streak — play as many as you like.

Streaks

Completing the daily run on a given UTC day extends your streak. Missing a day breaks it. If life gets in the way, a free streak rescue is available roughly once every 30 days.

Hints

Hints reveal one letter mapping at the cost of points. The cost and number of available hints depend on difficulty.

Account & data

Do I need to sign in?

No. The game works fully offline as a single-player puzzle. Signing in unlocks: cloud-synced progress across devices, public leaderboards, and durable streak state.

How do I delete my account?

In the app, open Settings → Account → Delete account. This permanently removes your Firebase account, private sync data, and leaderboard rows. The action cannot be undone.

How do I change my display name?

Settings → Account → Display name. The new name appears on leaderboards within a few seconds.

I deleted the app — is my progress gone?

Local progress (run history, settings, achievements) is removed with the app. Cloud-synced progress (streaks, leaderboard standing) is preserved if you signed in; sign in again on a fresh install and your account state will restore.

Common issues

The leaderboard isn't updating

Leaderboard writes need a network connection. If you played offline, your run is queued locally and uploads next time the app is open with internet access. The first run of a UTC day is always the one that counts; runs 2–4 award smaller scores; the 5th and beyond don't count toward leaderboards (but you can still play them for fun).

"Sign in with Apple/Google" failed

Make sure you're signed into iCloud / a Google account on the device, then try again. If the issue persists, switch to anonymous sign-in (the game still saves locally) and email us with the device model and OS version.

The puzzle has no solution / the answer doesn't match

Each puzzle is generated deterministically and is fully solvable. If you believe you've found a counter-example, please email us with the seed (visible on the run results screen) and a screenshot — that's enough information to reproduce the run exactly.

I want to suggest a quote / inscription / language

Suggestions are welcome. Email m.gokhanoguz2@gmail.com with the text and a source if applicable.

Bug reports

The most useful bug reports include:

  • Device and OS version (e.g. iPhone 15, iOS 18.3)
  • App version (visible in Settings → About)
  • What you did, what you expected, what happened
  • If reproducible: the run seed and difficulty
  • Screenshot or screen recording, if relevant
